The puprose of this bot is to fetch data from zillow.com and save it inside a google sheet.

The data fetched is:
    1. Property Address
    2. Property Rent
    3. Ad Link

The data is fetched using BeautifulSoup4

The data is then automatically uploaded to google forms and later converted to a google sheet.

This project was created for a Turkish resident. So if you want to use this, make sure you replace some xpaths
in Automation.py > fill_details and Automation > create_google_Sheet
